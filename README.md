# 🌿 Le Grand Vert — Dossier Netlify

## Contenu du dossier
```
legrandvert/
├── index.html       ← Site principal
├── wifi.png         ← QR Code WiFi (à ajouter)
└── netlify.toml     ← Configuration Netlify
```

## ⚠️ À faire avant de déployer
**Ajouter votre QR code WiFi :**
1. Générez votre QR code WiFi sur https://qr.io ou https://www.qrcode-monkey.com/
2. Paramètres : Réseau = `Le Grand Vert`, Mot de passe = votre mot de passe WiFi, Type = WPA
3. Téléchargez l'image en PNG
4. Renommez-la `wifi.png`
5. Placez-la dans ce dossier

## 🚀 Déploiement sur Netlify

### Option 1 — Drag & Drop (le plus simple)
1. Allez sur https://app.netlify.com
2. Créez un compte gratuit
3. Faites glisser **tout ce dossier** dans la zone de dépôt
4. Votre site est en ligne en 30 secondes !

### Option 2 — Via GitHub
1. Créez un repo GitHub avec ce dossier
2. Sur Netlify → "New site from Git"
3. Connectez votre repo
4. Deploy !
